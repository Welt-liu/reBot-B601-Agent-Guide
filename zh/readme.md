# reBot B601-RS & B601-DM AGENT Skills

本 AGENT 文档目标是协助用户初步排查机械臂遇到的问题，并且协助用户安装所需环境。

让 agent 来帮助你初始化机械臂，复制以下内容发送给 agent：

```
请参考 AGENTS.md（https://github.com/Welt-liu/reBot-B601-RS-Skills/blob/main/zh/AGENTS.md）中的流程，帮助用户完成新机械臂的初始化。
```
