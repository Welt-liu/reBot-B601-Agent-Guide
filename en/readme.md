# reBot B601-RS & B601-DM AGENT Skills

This AGENT documentation assists users with initial troubleshooting for the robotic arm and helps with environment setup.

To let the agent help you initialize the robotic arm, copy the following prompt and send it to your agent:

```
Please follow the workflow in AGENTS.md (https://github.com/Welt-liu/reBot-B601-RS-Skills/blob/main/en/AGENTS.md) to help me initialize the robotic arm.
```
